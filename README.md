# 知乎自动点赞和喜欢

在知乎个人/机构回答界面自动批量点赞和喜欢，支持可调间隔、一键开始/停止，自动滚动加载并处理到底部停止。

**作者**：amoorzheyu

## 功能说明

- **自动点赞**：自动点击当前页面中未点赞的「赞同」按钮
- **自动喜欢**：自动点击未点「喜欢」的内容
- **自动滚动**：处理完当前可见内容后自动向下滚动，加载更多
- **可调间隔**：可设置每次点击的间隔时间（毫秒），默认 2000ms，设置会保存到本地
- **一键控制**：通过悬浮面板开始/停止任务，到达页面底部自动停止

## 适用页面

脚本在以下知乎页面生效：

- `https://www.zhihu.com/people/*/answers`（个人回答）
- `https://www.zhihu.com/people/*/posts`（个人想法）
- `https://www.zhihu.com/org/*/answers`（机构回答）
- `https://www.zhihu.com/org/*/posts`（机构想法）

## 安装与使用

### 安装

1. 浏览器需已安装用户脚本扩展之一：
   - [Tampermonkey](https://www.tampermonkey.net/)（油猴）
   - [Script Cat](https://scriptcat.org/)（脚本猫）
2. 打开 `知乎自动点赞和喜欢.user.js`，按扩展提示安装脚本；或复制脚本内容，在扩展中新建脚本并粘贴保存

### 使用

1. 打开上述任一知乎页面（如某用户的「回答」或「想法」列表）
2. 页面加载完成后，右上角会出现「知乎自动点赞」控制面板
3. 可选：在「点击间隔(ms)」中修改间隔（如 2000 表示每 2 秒一次）
4. 点击「开始」按钮，脚本会依次点赞、喜欢，并自动滚动；点击「停止」可随时中断
5. 滚动到页面底部且无新内容可处理时，会显示「已完成」并自动停止

## 技术说明

- **@grant**：none，不请求额外浏览器权限
- **存储**：仅使用 `localStorage` 的 `zhihu-auto-interval` 保存点击间隔设置

## 注意事项

- 请合理设置间隔，避免操作过于频繁
- 脚本仅辅助操作，使用需遵守知乎社区规范与使用条款
